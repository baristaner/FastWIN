# Installation
1. Clone the repository:
```bash
git clone https://github.com/baristaner/fastwin.git
```

2. Change to the project directory:
```bash
cd src/fastwin
```

3. Build the project:
```bash
dotnet build 
```

4. Run the project:
```bash
dotnet run 
```

5-Run The Migrations
```
dotnet ef migrations add MigrationName
```
